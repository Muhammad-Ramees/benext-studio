# BeNext Studio — Website

Next.js 14 website for BeNext Studio, Dubai.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Before Launch — Update These:

1. **WhatsApp number** — search `+971542711288` in components/Navbar.js and components/ContactCTA.js
2. **Email** — update `info@benextstudio.com` in ContactCTA.js
3. **Product links** — update `href="#"` in Products.js with real URLs
4. **SEO metadata** — update title/description in app/layout.js

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Syne (display font) + Instrument Sans (body font)
- Zero dependencies beyond Next + Tailwind

## Sections
- Navbar (sticky, mobile menu)
- Hero (animated, stats)
- Ticker (scrolling tags)
- Services (4 service cards)
- Products (Malaeb + Academy OS featured)
- Why Us (6 differentiators)
- Process (4-step)
- Contact CTA (WhatsApp + email)
- Footer
